# Admin module - Ngày 1

Phần này là khung ban đầu cho module quản trị của đồ án Flutter LaptopAI.

## Phạm vi đã tạo

- Layout dùng chung cho admin: `lib/shared/layouts/admin_layout.dart`
- Controller khung cho các chức năng admin
- Màn hình placeholder cho dashboard, sản phẩm, danh mục, đơn hàng, đánh giá, thống kê AI và cài đặt
- Route cho phần quản trị, bao gồm `adminReviews`

## Các file Hiền phụ trách chính

```txt
lib/features/admin/
lib/shared/layouts/admin_layout.dart
lib/routes/app_routes.dart
lib/routes/app_pages.dart
```

## Ghi chú

Ngày 1 chỉ dựng khung, chưa kết nối API thật. Các ngày sau sẽ hoàn thiện dashboard, sản phẩm, danh mục, đơn hàng và thống kê.
